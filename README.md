# phalcon-auth
Simple Phalcon Micro Auth App

### get
`git clone https://github.com/matiaspub/phalcon-auth.git`

### setup
`cp .env-example .env`
`cp users/config-example.ini config.ini`

### run
`docker-compose up -d`

### seed
`docker-compose exec tools phalcon migration run`

### browse
[open](http://localhost)
